Core Features

Built for Production

Every feature implemented with real-world patterns and clean architecture.

🔐

JWT Authentication

Stateless token-based auth with BCrypt password hashing. Separate USER and ADMIN roles with method-level security.

📦

Inventory Control

Real-time stock tracking with pessimistic locking to prevent race conditions under concurrent requests.

🛒

Order Lifecycle

State machine enforcing valid transitions: PENDING → CONFIRMED → SHIPPED → DELIVERED. Terminal states protected.

🏗️

Clean Architecture

Domain-first package structure. Controller → Service → Repository separation. DTOs with validation on all inputs.

🧪

Fully Tested

Unit tests with Mockito for service layer. Integration tests via Testcontainers with a real PostgreSQL instance.

🐳

Docker Ready

Multi-stage Dockerfile and docker-compose for one-command deployment. HikariCP connection pool configured.
